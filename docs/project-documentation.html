JMX Forge Documentation

Generated on 2026-05-05.

New Feature: AI-Based Correlation

  • Engine scans each response body/header/cookie candidate across all captured requests.
  • Each candidate is checked against subsequent request URL/body/header usage.
  • Correlations are scored by confidence and presented with explanation reason.
  • Only high-confidence dynamic fields are added to JMX extractors.

New Feature: JTL + XML Analytics Dashboard

  • Uploads: one JTL file and one XML/JMX file.
  • Outputs: start/end time, duration, throughput, errors, average/min/max response times.
  • Percentiles: p50, p80, p90, p95, p99.
  • Peak window insights: peak started at and peak users.
  • Top-label breakdown with per-label percentile/error profile.

APIs Added

  • POST /api/analyze-results (multipart: jtl_file, xml_file)
  • GET /api/health
  • GET /api/jobs
